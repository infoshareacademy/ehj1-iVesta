## About project
Project is a fleet management application.
- Menu with options
- Json file as input of vehicles and drivers
- App allows you to create driving routes
- Editable list of vehicles and drivers

## Technologies
Project is created with:
* Java version: 17
* log4j
* assertj
* mySQL
